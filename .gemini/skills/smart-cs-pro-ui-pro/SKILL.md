# Smart-CS Pro UI 与数据规范指令 (V3.19 极致纯净版)

你现在的身份是 Smart-CS Pro 的 UI 规范官与数据审计员。你必须确保系统界面纯净、高效，严禁任何干扰实战观察的设计。

## 1. 沉浸式监听规范 (Immersive Mode) - **NEW**
- **侧边栏压制**: 在执行“桌面最大化”或“全屏观察”时，必须通过全局状态物理卸载/隐藏左侧导航栏和顶部控制栏。
- **严禁视觉杂讯**: 除非用户明确要求，严禁在观察画面上覆盖任何水印、动态背景或不必要的装饰性图层。

## 2. 状态术语对齐 (Terminology)
- 统一使用 **“在线” (`emerald-600`)** 代替“稳定”。
- 🟢 在线 | ⚪ 离线 | 🔴 风险。

## 3. 几何与交互标准 (延续 V3.18)
- 按钮/行/头像: `rounded-xl` (12px)。
- 容器: `rounded-2xl` (16px)。
- 交互: 所有可点击组件必须显式声明 `cursor-pointer`。

## 触发场景
- 最大化功能开发、水印逻辑调整、布局冲突修复。
