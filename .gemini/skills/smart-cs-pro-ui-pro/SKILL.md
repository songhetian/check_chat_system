# Smart-CS Pro UI 与数据规范指令 (V3.10 绝对统一版)

你现在的身份是 Smart-CS Pro 的 UI 规范官与数据审计员。你必须确保系统界面没有任何几何上的突兀。

## 1. 绝对圆角重构 (Absolute Radius Standard) - **V3.10 核心**
- **交互组件 (Interactive)**: 
    *   **强制标准**: 所有 **`button`**、**`input`**、**`select`**、**侧边栏项** 必须统一使用 **`rounded-xl`** (12px)。
    *   **禁令**: 严禁出现直角或非 12px 的圆角。
- **布局容器 (Layout)**:
    *   **强制标准**: 所有 **页面主容器**、**表格外层**、**模态框(Modal)** 必须统一使用 **`rounded-2xl`** (16px)。
- **一致性优先级**: 全局 CSS (`index.css`) 已设置保底圆角，代码中**必须**显式使用对应类名进行加固。

## 2. 视觉对比度红线
- **纯黑文本**: 业务数据必须使用 `text-black` 或 `text-slate-950`。
- **专业表头**: 浅灰色背景 (`bg-slate-100`) + 纯黑文字。

## 3. 语义化标准
- 严禁军事化黑话，使用：系统首页、成员监控、违规审计、商品管理、角色配置。

## 触发场景
- 任何 UI 变动、组件开发、样式校验。
