# 📖 Smart-CS Pro 战术级系统操作手册

## 第一部分：环境部署 (运维必读)

### 1. 快速初始化
1.  **编辑配置**：打开 `SmartCS_Desktop/smart-cs-pro/.env`，根据实际情况配置 MySQL 地址、Redis 地址以及本地服务器 IP。
2.  **一键安装**：
    ```bash
    python core_engine/init_system.py
    ```
    该脚本会自动安装库、建立数据库表、同步 schema。

### 2. 服务器配置 (场景 B)
*   若要多机共享 AI，请在服务器上运行：`set OLLAMA_HOST=0.0.0.0` 后启动 Ollama。

---

## 第二部分：坐席端操作 (员工篇)

### 1. 启动与仪式感登录
*   **动作**：运行程序，输入操作员编号。
*   **提示**：请戴上耳机。系统会进行“神经链路初始化”语音播报，待进度条满 100% 后自动进入系统。

### 2. 灵动岛交互
*   **移动**：点击灵动岛 Header 区域可全局拖拽。
*   **展开**：点击右侧“指纹/箭头”图标，或双击切换至大屏。
*   **智能纠偏**：
    1.  开启“AI 优化模式”开关。
    2.  听到“Ping”声表示话术已就绪。
    3.  在对话框直接 `Ctrl+V`。
*   **新兵求助**：若不熟悉业务，开启“新兵带教模式”，随时查看琥珀色卡片的 SOP 引导。

### 3. 应对预警
*   **普通报警**：灵动岛闪红，播放警告音。
*   **特级警报**：全屏红框闪烁，播放重低音，并有中文语音指令。请立即停止违规输入！

---

## 第三部分：主管端操作 (管理者篇)

### 1. 实时指挥
*   **首页看板**：查看坐席矩阵。绿色脉冲表示该坐席扫描器活跃，红色闪烁表示正在违规。
*   **远程接管**：点击坐席卡片上的“指令”图标，可下发全屏强制通知。

### 2. 取证与复盘
*   **紧急响应队列**：进入“违规记录”页。
*   **AI 提炼**：点击详情页的“AI 智能提炼”，快速查看对话核心矛盾。
*   **视频回放**：点击“视频取证”，回看违规发生前后的屏幕录像。

### 3. 数据操作
*   **导出**：点击右上角“导出战术详表”，一键获取本部门全量 Excel 审计数据。

---

## 第四部分：总部端操作 (高管篇)

### 1. 权限管理
*   进入“公司成员矩阵”。
*   点击用户下方的角色按钮（AGENT / ADMIN / HQ）可实时调岗。
*   **注意**：调岗后该员工会收到语音提醒并被强制退出系统。

### 2. 策略下发
*   进入“全局 AI 策略”。
*   在“语音协议库”中修改警告词条。修改后点击“测试播报”试听，确认无误后点击“全员同步”。

---

## 五、 故障处理
1.  **听不到声音**：检查 `.env` 中的音效配置，确保系统没有静音。
2.  **不弹出画像**：检查聊天窗口是否在前台，标题是否包含“微信”或“钉钉”。
3.  **连接失败**：检查服务器 IP 是否在 `.env` 中设置正确，后端 8000 端口是否开放。
